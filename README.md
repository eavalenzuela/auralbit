# auralbit

A small music manager + player. Intended to live in 1/4 of the screen. Think GUI version of cmus, rather than Rhythmbox.

## Features

* Support for common music filetypes
* Playlists + import/export
* 'Sync playlist to' copies playlist file and syncs music folders+files to remote repor (e.g. phone via USB)
* Colored box visualizer when playing
* Natively dark-themed
